# Extious - Online Resume

A minimal online resume with view, edit, and export features.

## Features

- View page `/`
- Edit page `/edit`
- Export to PDF and PNG

## Tech

- Next.js 14
- TypeScript
- Tailwind CSS
- html2canvas + jsPDF

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Customize

Edit `types/cv.ts` to change data.

## Build

```bash
npm run build
npm start
```
